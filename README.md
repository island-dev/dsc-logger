# dsc-logger

Discordの過去のメッセージログをjsonファイルとして保存できるツール

## How to use

Clone this repository

```bash
git clone https://github.com/island-dev/dsc-logger
```

And Run!

```bash
cd dsc-logger
touch .env
deno run -Ar index.ts
```

## Mode Description

1 >> 特定のチャンネルIDのメッセージを指定した数、保存できます

2 >> 特定の人物のメッセージだけを保存(WIP)

3 >> トークン設定
